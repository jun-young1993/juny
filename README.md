# Setup
## Prerequisite
- Node.js 18.17 or latest
## Installation
### 1. Fork And Clone this repo
#### Version Sync
1. git add remote
```shell
git remote add blog https://github.com/jun-young1993/nextjs-github-blog.git
```
2. git fetch blog
```shell
git fetch blog
```
3. git merge remote
```shell
git merge blog/main
```
4. From now on...
```shell
git fetch blog 
...
git merge blog/main
```
### 2. package install
```shell
npm install
```
### 3. config
```shell
cp ./github.blog.config.ts.example cp./github.blog.config.ts
```
